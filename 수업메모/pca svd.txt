1. PCA (Principal Component Analysis, 주성분 분석)
1-1. 목적

PCA는 데이터의 분산(variation)을 최대한 보존하면서 차원을 줄이는 방법이에요.
즉, 많은 변수(특징) 중에서 가장 중요한 방향을 찾아서 데이터를 요약합니다.

예: 100개의 변수로 이루어진 데이터 → 2~3개의 주요 구성 요소로 줄여 시각화 가능

핵심: 정보 손실을 최소화하면서 데이터 구조를 단순화
-------------------------------------------------------------
1-2. 직관적 이해

데이터가 n차원 공간에 흩어져 있다고 생각해요.

이 데이터를 가장 잘 설명하는 직선/평면/초평면을 찾습니다.

1차원으로 줄이면 가장 데이터가 퍼져 있는 방향을 찾는 것

2차원으로 줄이면 가장 데이터가 퍼져 있는 평면을 찾는 것

이 방향을 **주성분(Principal Components)**이라고 부릅니다.

쉽게 말하면, “데이터가 가장 많이 늘어져 있는 방향이 뭘까?”를 찾는 과정이에요.
핵심 단계: 중앙 정렬 → 공분산 계산 → 고유값 분해 → 상위 k개 선택

=============================================

2. SVD (Singular Value Decomposition, 특이값 분해)
2-1. 목적

SVD는 어떤 행렬 A를 세 개의 행렬로 분해하는 방법이에요.

쉽게 말하면, 행렬을 회전 + 스케일 + 회전으로 쪼개는 과정이에요.

2-2. 직관적 이해

데이터 행렬을 회전과 스케일링으로 표현

중요한 축(특이값이 큰 것) → 데이터 정보가 많이 들어 있음

차원 축소 가능 → 상위 k개 특이값만 남겨서 근사

PCA와 밀접한 관계가 있어요. 사실 PCA는 SVD를 사용해서 더 효율적으로 계산 가능








PCA와 SVD 관계: SVD로 PCA 계산 가능. 훨씬 안정적이고 빠름.
